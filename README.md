## Gerenciador de projeto/trabalho feito para a 2PI Studio como projeto avaliativo para a disciplina de Estagio Obrigatório do curso de Sistemas de Informação da Universidade Federal do Acre

Para executar localmente é necessário clonar este repositório e:
### instalar as dependencias via:
`npm install`
### e então executar, na raiz do projeto, utilizando:
`npm run dev`
